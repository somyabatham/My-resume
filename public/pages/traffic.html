TRAFFIC LIGHT

Creating a Traffic Light Using HTML, CSS, and JavaScript

  • A traffic light project involves creating a visual representation of a traffic light and programming it to simulate the changing of the lights.
  • The basic structure of the traffic light can be created using HTML, with each light (red, yellow, and green) represented by a <div> element.
  • CSS is used to style the appearance of the traffic light, including the size, shape, and color of the lights.
  • JavaScript is then used to control the logic of the traffic light, changing the opacity of the lights to simulate the traffic light sequence.

HTML Structure

  1. The HTML structure includes a <div> element with the ID "traffic-light" to contain the traffic light.
  2. Inside this container, there are three <div> elements, each representing a light (red, yellow, and green).
  3. The HTML also includes a <script> element to link the JavaScript file.

CSS Styling

  • The CSS styles the traffic light container and the individual lights.
  • The traffic light container is given a dark background, black border, padding, and a circular border-radius to create the overall appearance.
  • Each light is styled with a specific background color (red, yellow, or green) and a circular border-radius to create the appearance of a light bulb.

JavaScript Logic

  • The JavaScript code defines a function called myFun() that controls the changing of the lights.
  • Inside this function, the code selects the individual light elements using their class names.
  • The function then sets the opacity of the lights in a specific sequence to simulate the traffic light cycle.
  • The initial state of the traffic light is set to have the green light turned on.
  • After a 3-second delay, the green light fades out, the yellow light turns on, and the red light turns on.
  • After another 3-second delay, the yellow light fades out, and the red light remains on.
  • Finally, after a third 3-second delay, the red light fades out, and the green light turns on again.
  • The function is then set to run continuously using setInterval() to create the ongoing traffic light cycle.
